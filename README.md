# Verilog Learning Series

Welcome to my **Verilog Learning Series**!

This repository documents my journey of learning **Verilog Hardware Description Language (HDL)** as part of my preparation for a career in **VLSI Design**.

Instead of simply studying Verilog, I believe the best way to learn is by **designing, simulating, documenting, and sharing** every project. This repository contains all the digital circuits I build, along with their Verilog source code, testbenches, simulation waveforms, and documentation.

#  Learning Approach

For every project, I follow the same workflow:

1. Learn the digital design concept
2. Derive the logic equations
3. Write the Verilog RTL
4. Create a testbench
5. Simulate and verify the design
6. Document the project
7. Upload to GitHub
8. Share my learning on LinkedIn
9. Create educational content for YouTube
